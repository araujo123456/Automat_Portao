Porteiro Eletrônico V2.4

Instalador local via navegador, sem GitHub Pages.

Como usar:
  1. Extraia a pasta inteira.
  2. Execute run-local.bat.
  3. Quando abrir o navegador, clique em Conectar e Gravar.

Importante: não abra este arquivo com duplo clique direto. Use o run-local.bat, que inicia em http://localhost:8000.

Antes de começar:
  • Use Google Chrome ou Microsoft Edge
  • Conecte o ESP32 por cabo USB
  • Feche o Monitor Serial e o PlatformIO
  • Se necessário, coloque a placa em modo boot
